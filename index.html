tagManager

Uses Bootstrap (2.2.2), Require jQuery (1.8.3+)
These are TagManagerCtrl's $scope tags : {{tag}}

The tag-manager directive creates client-side tag-manager component. The directive creates an input field that reacts with comma characters between each tag. Each comma separated tag is stored in an array in tag-manager self $scope. This tag array can be accessed outside by mapping a value to ng-model value.
Backspace key is handled to load previous tag in tag editor when tag field is empty.

Example : <tag-manager ng-model="tags">

From module: angular-board.tagManager
Futures improvements :
  • handle typeahead tag completion based on an ajax request.
  • tags can be storred in a list or a set in order to disallow duplicated tags
